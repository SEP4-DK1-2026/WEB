# SEP4 Weatherstation Frontend

The frontend of the SEP4 project is built using React and Vite.  
The goal is to fetch data from a microservice that connects to data from the other modules in the system (MAL, IoT).

## Project structure

Using 'Bulletproof React' as a baseline. See [project-structure.md](/project-structure.md)

## Docker configuration

Using a compose based devcontainer and app container. See [README.Docker.md](/README.Docker.md)
